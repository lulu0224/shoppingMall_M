移动端（共7个页面，已完成7个页面）

一、“首页页面”index.html说明：
 
1、banner下面的菜单栏中的“商品分类”，点击后跳转到->商品分类页面（goods_style.html）；
2、主体中最后一层的热门推荐下面，点击第一个商品的图片，点击后跳转到->商品详情页面（goods_info.html）；
3、底部footer菜单，点击“首页”,跳转到->首页页面（index.html）；点击“购物车”,跳转到->购物车页面（cart.html）；
    点击“我的”,跳转到->我的页面（mine.html）；


二、“购物车页面”cart.html说明：

1、点击顶部header里的右箭头“<”,跳转到->首页页面（index.html）；
2、点击“去结算”，跳转到->确认订单页面（sure_order.html）；
3、底部footer菜单，点击“首页”,跳转到->首页页面（index.html）；点击“购物车”,跳转到->购物车页面（cart.html）；
    点击“我的”,跳转到->我的页面（mine.html）；


三、“我的页面”mine.html：

1、点击顶部header里的右箭头“<”,跳转到->首页页面（index.html）；
2、底部footer菜单，点击“首页”,跳转到->首页页面（index.html）；点击“购物车”,跳转到->购物车页面（cart.html）；
    点击“我的”,跳转到->我的页面（mine.html）；


四、“商品分类页面”goods_style.html说明：

1、点击顶部header里的右箭头“<”,跳转到->首页页面（index.html）；
2、点击内容区右边的第一层“更多>”，跳转到->商品列表页面（goods_list.html）；


五、“商品列表页面”goods_list.html说明：

    点击“我的”,跳转到->我的页面（mine.html）；


六、“商品详情页面”goods_info.html说明：

2、底部footer菜单，点击“购物车”,跳转到->购物车页面（cart.html）；
